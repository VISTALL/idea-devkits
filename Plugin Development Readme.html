IntelliJ IDEA Plugin Development Readme

This file gives a quick overview of the contents of the IntelliJ IDEA Plugin Development Package.

Documentation and Source Code

The package includes the OpenAPI documentation and the source code of the IntelliJ IDEA OpenAPI.

The package also includes the following documents:

  • Description of the overall plugin structure and the plugin.xml format
  • Overview of the IntelliJ IDEA action system
  • Developing custom language plugins for IntelliJ IDEA 5.x

Additional documentation, including a plugin development tutorial, is available on the JetBrains Web site.

Example Plugins

The example plugins are small projects which demonstrate the usage of specific aspects of the IntelliJ IDEA OpenAPI and can serve as basis for your plugin projects.

  • Plugin sample demonstrates the smallest possible IntelliJ IDEA plugin, which implements the ApplicationComponent and ProjectComponent interfaces.
  • Actions sample demonstrates adding actions to the IDEA toolbar and menu bar, through AnAction and related classes.
  • ToolWindow sample demonstrates adding a custom tool window to the IDEA user interface, through ToolWindow and related classes.
  • VFS sample demonstrates working with the IDEA Virtual File System, through VirtualFile and related classes.
  • Comparing References sample is a sample code inspection which detects potentially incorrect uses of the == and != operators and suggests to replace them with .equals() method calls. The sample also demonstrates the usage of the IDEA PSI (Program Structure Interface) API.
  • Conditional Operator Converter sample is a sample intention action which allows to replace usages of the ternary operator (? :) with corresponding 'if' statements. The sample also demonstrates the usage of the IDEA PSI (Program Structure Interface) API.

Plugin Source Code

The package includes the source code for a number of plugins supplied with IntelliJ IDEA.

Source Control Integration

  • Visual SourceSafe integration
  • StarTeam integration
  • Subversion integration
  • Perforce integration

Application Server Integration

  • WebLogic integration
  • Tomcat integration
  • Debugging support for JSR-45 compatible application servers

Other Plugins

  • Plugin Development Kit
  • Images support plugin
  • InspectionGadgets plugin (implements more than 400 code inspections)
  • IntentionPowerPack plugin (implements more than 40 intention actions)
  • J2ME development support plugin
  • JavaScript support plugin (demonstrates the use of IntellJ IDEA 5.x Language API)
  • KlassMaster stacktrace unscramble plugin

Development Tools

The tools/jflex directory includes the patched version of JFlex 1.4.1, which can be used for compiling custom language lexers. This version of JFlex is required for recompiling the lexer of the JavaScript support plugin included in the distribution.

Start Creating Your Own Plugin

An extensive tutorial for developing plugins with IntelliJ IDEA 5.1 is available on JetBrains Web site.

Additional Plugin Development Resources

  • You can discuss any issues related to the IntelliJ IDEA OpenAPI with the JetBrains team and other plugin developers in the jetbrains.intellij.openapi newsgroup or in the corresponding Web forum.
  • Once your plugin is ready, you can upload it to the IntelliJ IDEA Plugin Repository.
